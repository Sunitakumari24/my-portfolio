# Sunita's Portfolio Website

A modern, responsive portfolio website showcasing web development skills, projects, and professional experience.

## 🎨 Features

- **Modern UI/UX Design**: Clean, gradient-based design with smooth animations
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Dark Mode**: Toggle between light and dark themes with smooth transitions
- **Interactive JavaScript Features**:
  - 📝 Typing animation effect on homepage
  - 🎯 Smooth scroll navigation with active section highlighting
  - 🎨 Custom cursor with follower effect
  - ⬆️ Scroll-to-top button
  - 🎭 Interactive hover effects on all cards
  - ✨ Scroll-triggered animations for sections
  - 📱 Mobile-responsive hamburger menu
  - 💫 Particle background animation
  - ⚡ Loading screen with spinner
  - 🎪 Timeline items animation
  - ✅ Form validation with error messages
  - 🎉 Success message notifications
  - 🌓 Theme persistence with localStorage
  - 🎬 Profile image slideshow
  - 🎨 Skill cards progressive animation
  - 🔍 Dynamic navbar on scroll
- **Interactive Sections**:
  - Homepage with animated background
  - About Me section
  - Skills showcase with colorful cards
  - Projects gallery with live demo links
  - Timeline-based Education & Experience sections
  - Certification showcase
  - Contact form with social media links

## 🚀 Getting Started

### Installation

1. Clone or download this repository
2. Open the project folder
3. Open `index.html` in your browser

### Local Development

Simply open `index.html` in any modern web browser:
- Chrome
- Firefox
- Safari
- Edge

### Deployment

You can deploy this portfolio on any of these platforms:
- **Netlify**: Drag and drop the folder
- **Vercel**: Import from GitHub
- **GitHub Pages**: Push to a repository and enable Pages
- **Any static hosting service**

## 🎨 Customization

### Changing Colors

Edit the CSS variables in `style.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... more variables */
}
```

### Adding Your Information

1. **Profile Picture**: Replace `image/image.png` with your photo
2. **Personal Info**: Update text content in `index.html`
3. **Projects**: Add/modify project cards in the Projects section
4. **Skills**: Update skills in the Skills section
5. **Links**: Update social media and project links

### Sections to Update

- **Hero Section**: Name, title, description
- **About**: Personal introduction
- **Skills**: Technical and soft skills
- **Projects**: Your work with images and links
- **Education**: Academic background
- **Experience**: Work experience
- **Certifications**: Professional certifications
- **Contact**: Email and social links

## 🌈 Color Scheme

The portfolio uses a modern purple-blue gradient theme:
- Primary: `#667eea` (Purple-Blue)
- Secondary: `#764ba2` (Deep Purple)
- Accent: `#f093fb` (Pink)

Dark mode automatically adjusts colors for better contrast.

## ✨ JavaScript Features Explained

### 1. **Typing Animation**
Homepage heading types out character by character for an engaging first impression.

### 2. **Custom Cursor Effect**
Interactive cursor with a follower element that responds to mouse movements.

### 3. **Scroll-to-Top Button**
Appears after scrolling down 300px, smoothly scrolls back to top on click.

### 4. **Active Navigation Highlighting**
Automatically highlights the current section in the navigation menu as you scroll.

### 5. **Form Validation**
Real-time validation for contact form with custom error messages and success notifications.

### 6. **Intersection Observer API**
Efficient scroll-triggered animations for sections, skills, and timeline items.

### 7. **Theme Persistence**
Dark/light mode preference saved in localStorage and restored on page reload.

### 8. **Dynamic Navbar**
Navbar changes shadow and padding based on scroll position for better UX.

### 9. **Particle Animation**
Dynamic background particles created and animated programmatically.

### 10. **Loading Screen**
Initial loading animation for better perceived performance.

### 11. **Progressive Skill Cards**
Skills animate in sequence with staggered delays for visual appeal.

### 12. **Interactive Hover Effects**
Enhanced hover states for cards, buttons, and interactive elements.
